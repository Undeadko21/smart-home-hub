# Установка Smart Kiosk на TrueNAS SCALE 25.10.1 Goldeye

## Быстрая установка через YAML (рекомендуется)

Этот способ установки использует нативный механизм TrueNAS SCALE 25.10.1 Goldeye для установки приложений через YAML-конфигурацию с последующей настройкой всех параметров (токены, URL, ключи API) через удобный веб-интерфейс приложения.

### Шаг 1: Откройте интерфейс Apps

1. Войдите в веб-интерфейс TrueNAS SCALE
2. Перейдите в раздел **Apps**
3. Нажмите **Manage Catalogs** → **Add Git Catalog**

### Шаг 2: Добавьте каталог приложения

Заполните форму добавления каталога:

| Поле | Значение |
|------|----------|
| **Name** | `smart-kiosk` |
| **Repository URL** | `https://github.com/Undeadko21/smart-home-hub.git` |
| **Branch** | `main` |

Нажмите **Save**.

### Шаг 3: Установите приложение

1. Перейдите в **Apps** → **Discover Apps**
2. Найдите **Smart Kiosk** в списке доступных приложений
3. Нажмите **Install**

### Шаг 4: Настройте через веб-интерфейс

После начала установки откроется форма настройки. Все параметры можно настроить сразу или изменить позже через веб-интерфейс самого приложения.

#### Основные настройки (обязательно)

| Параметр | Описание | Пример |
|----------|----------|--------|
| **Порт приложения** | Порт для доступа к веб-интерфейсу | `8080` |
| **Путь хранения данных** | Директория на хосте для БД, логов и кэша | `/mnt/data/smart-kiosk` |
| **Лимит CPU** | Максимальное количество CPU ядер | `1.0` |
| **Лимит памяти** | Максимальный объем RAM | `512M` |

#### Интеграция с Home Assistant (опционально)

| Параметр | Описание |
|----------|----------|
| **Home Assistant URL** | URL вашего сервера HA (например, `http://192.168.1.100:8123`) |
| **Home Assistant Token** | Long-Lived Access Token из профиля пользователя HA |

> **Как получить токен Home Assistant:**
> 1. Откройте Home Assistant
> 2. Профиль пользователя → внизу страницы **Long-Lived Access Tokens**
> 3. Нажмите **Create Token**, дайте имя (например, `smart-kiosk`)
> 4. Скопируйте токен и вставьте в поле настройки

#### Настройки AI ассистента (опционально)

| Параметр | Описание |
|----------|----------|
| **DeepSeek API Key** | API ключ для сервиса DeepSeek (AI ассистент) |

> **Как получить ключ DeepSeek:**
> 1. Зарегистрируйтесь на [platform.deepseek.com](https://platform.deepseek.com)
> 2. Перейдите в раздел API Keys
> 3. Создайте новый ключ и скопируйте его

#### Дополнительные настройки Docker Compose

| Параметр | Рекомендуемое значение | Описание |
|----------|----------------------|----------|
| **Имя проекта** | `smart-kiosk` | Имя для контейнеров Docker |
| **Политика перезапуска** | `Unless Stopped` | Автоматический перезапуск при сбоях |
| **Проверка здоровья** | `Включено` | Мониторинг состояния контейнера |
| **Драйвер логов** | `JSON File` | Стандартный драйвер логирования |
| **Макс. размер логов** | `10m` | Ротация логов при достижении размера |
| **Кол-во файлов логов** | `3` | Количество сохраняемых файлов логов |

### Шаг 5: Завершите установку

Нажмите **Install** для завершения установки. Процесс займёт 2-5 минут.

После установки приложение будет доступно по адресу:
```
http://<IP-адрес-TruNAS>:8080
```

## Управление приложением

### Просмотр статуса и логов

1. Перейдите в **Apps** → **Installed Apps**
2. Найдите **Smart Kiosk**
3. Нажмите на название приложения для просмотра:
   - Статуса контейнеров
   - Использования ресурсов
   - Логов в реальном времени

### Изменение настроек

1. В разделе **Installed Apps** найдите **Smart Kiosk**
2. Нажмите **Edit** (или три точки → **Edit**)
3. Измените необходимые параметры (токены, порты, лимиты)
4. Нажмите **Update** для применения изменений

### Обновление приложения

1. Перейдите в **Apps** → **Installed Apps**
2. Найдите **Smart Kiosk**
3. Если доступна новая версия, появится кнопка **Update**
4. Нажмите **Update** для обновления до последней версии

### Остановка/Запуск

- **Остановить**: Три точки → **Stop**
- **Запустить**: Три точки → **Start**
- **Перезапустить**: Три точки → **Restart**

### Удаление приложения

1. Перейдите в **Apps** → **Installed Apps**
2. Найдите **Smart Kiosk**
3. Нажмите три точки → **Delete**
4. Подтвердите удаление

> **Внимание**: При удалении можно выбрать сохранение или удаление данных приложения.

## Решение проблем

### Приложение не запускается

1. Проверьте логи в интерфейсе Apps
2. Убедитесь, что выбранный порт не занят другим приложением
3. Проверьте права доступа к директории данных

### Ошибки подключения к Home Assistant

- Проверьте правильность URL (должен быть доступен из сети TrueNAS)
- Убедитесь, что токен действителен и не истёк
- Проверьте настройки firewall Home Assistant

### AI ассистент не работает

- Проверьте наличие и действительность API ключа DeepSeek
- Убедитесь, что TrueNAS имеет доступ в интернет
- Проверьте лимиты использования API ключа

## Технические характеристики

- **Порт по умолчанию**: 8080
- **Лимит CPU**: 1.0 ядра
- **Лимит памяти**: 512 МБ
- **Хранилище**: Host path binding (`/mnt/data/smart-kiosk`)
- **Минимальная версия TrueNAS SCALE**: 25.10.1 Goldeye
- **Тип установки**: Helm chart (Kubernetes)

---

## Переменные окружения

Все настройки применяются через переменные окружения контейнера:

| Переменная | Описание |
|------------|----------|
| `DATA_DIR` | Директория данных внутри контейнера (`/app/data`) |
| `HA_URL` | Home Assistant URL |
| `HA_TOKEN` | Home Assistant Long-Lived Access Token |
| `DEEPSEEK_KEY` | DeepSeek API Key |

---

**Поддержка**: Версия Kubernetes >= 1.25.0, TrueNAS SCALE 25.10.1+
