# Smart Kiosk - Home Assistant Add-on

Веб-приложение для умного дома с интеграцией Home Assistant и AI-ассистентом на базе DeepSeek.

## Возможности

- **Панель управления (киоск)** - удобный веб-интерфейс для управления устройствами
- **Интеграция с Home Assistant** - управление устройствами через API Home Assistant
- **AI-ассистент** - обработка естественного языка через DeepSeek API
- **Кэширование ответов** - оптимизация запросов к AI
- **Очередь уведомлений** - поддержка различных провайдеров уведомлений
- **Защита от перегрузки** - rate limiting для API endpoints

## Установка

### Через HACS (рекомендуется)

1. Откройте Home Assistant
2. Перейдите в **HACS** → **Add-ons**
3. Найдите **Smart Kiosk** в списке доступных дополнений
4. Нажмите **Install**
5. После установки перейдите в **Настройки** → **Дополнения** → **Smart Kiosk**
6. Настройте параметры и нажмите **Start**

### Ручная установка через Supervisor

1. Скопируйте папку `hassio-addon` в `/addons` на вашем сервере Home Assistant
2. В Home Assistant перейдите в **Настройки** → **Дополнения**
3. Нажмите **⋮** (три точки) → **Проверить обновления**
4. Найдите **Smart Kiosk** в списке локальных дополнений
5. Установите и запустите дополнение

## Настройка

### Основные параметры

| Параметр | Описание | Обязательно |
|----------|----------|-------------|
| `ha_url` | URL вашего сервера Home Assistant | Нет |
| `ha_token` | Long-Lived Access Token для Home Assistant | Нет |
| `deepseek_key` | API ключ для DeepSeek (AI ассистент) | Нет |

### Получение токена Home Assistant

1. Откройте профиль пользователя в Home Assistant
2. Прокрутите вниз до раздела **Long-Lived Access Tokens**
3. Нажмите **Create Token**
4. Введите имя токена (например, "Smart Kiosk")
5. Скопируйте токен и сохраните его в настройках дополнения

### Получение API ключа DeepSeek

1. Зарегистрируйтесь на https://platform.deepseek.com/
2. Перейдите в раздел API Keys
3. Создайте новый ключ
4. Скопируйте ключ и сохраните в настройках дополнения

## Использование

После запуска дополнения:

1. Откройте боковую панель Home Assistant
2. Найдите пункт **Smart Kiosk** (с иконкой киоска)
3. Откройте панель управления для доступа к интерфейсу

Или перейдите по адресу: `http://<your-ha-ip>:8080`

## API

Дополнение предоставляет следующие endpoints:

- `GET /` - Веб-интерфейс
- `GET /api/health` - Проверка состояния
- `POST /api/config` - Обновление конфигурации
- `POST /api/ha/call_service` - Вызов сервиса Home Assistant
- `GET /api/ha/entities` - Получение списка устройств
- `GET /api/ha/entity/{entity_id}` - Получение состояния устройства
- `GET /api/ha/services` - Получение списка сервисов
- `POST /api/ai/query` - Запрос к AI-ассистенту
- `POST /api/notify/test` - Тестовое уведомление

## Хранение данных

Дополнение хранит данные в директории `/data`:

- `app.db` - база данных SQLite (конфигурация, кэш AI, очередь уведомлений)
- `logs/app.log` - лог-файл приложения
- `cache/` - кэш файлов

## Требования

- Home Assistant 2023.1.0 или выше
- Доступ в интернет (для работы AI-ассистента)
- Свободное место на диске: ~100 МБ

## Поддерживаемые архитектуры

- AMD64 (x86_64)
- ARMv7
- ARM64 (AArch64)
- i386

## Логи

Для просмотра логов дополнения:

1. Перейдите в **Настройки** → **Дополнения** → **Smart Kiosk**
2. Откройте вкладку **Логи**

Или через SSH:
```bash
docker logs addon_smart_kiosk
```

## Безопасность

- Дополнение работает от непривилегированного пользователя
- Используется Home Assistant Ingress для безопасного доступа
- Rate limiting защищает API от перегрузки
- Токены и ключи хранятся в зашифрованном хранилище Home Assistant
